Darpan helps users set up reconciliation runs quickly, compare operational data, and review the evidence behind every difference.

Everything in the product points toward that run workflow. Connections, schemas, RuleSets, saved runs, generated outputs, and Ask Darpan navigation exist so a user can move from setup to a repeatable reconciliation run without hunting through configuration screens.
