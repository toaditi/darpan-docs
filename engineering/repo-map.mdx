## Boundary rule

Do not implement or extend custom Darpan UI in backend `screen/**`, `template/**`, or `theme-library/**`. The UI belongs in `toaditi/darpan-ui`. Add backend service contracts in `toaditi/darpan` when the UI needs data, then consume them from the Vue app.
