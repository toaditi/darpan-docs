## SFTP

Open **Ask Darpan** and search for **Open SFTP Servers**. For an existing record, search by the SFTP description, host, or username. Check host, port, username, authentication method, directory path, and network access. Confirm whether the server expects a private key, password, or another authentication method.

## NetSuite

Open **Ask Darpan** and search for **Open NetSuite Settings**. Check account target, role permissions, token validity, integration status, script deployment, and SuiteQL field names.

## AI providers

Open **Ask Darpan** and search for **Open AI Settings**. Check the provider record, credential reference, model availability, and feature-specific limits.

## Safe debugging

Capture error codes and messages, but redact secrets and customer-specific identifiers before sharing externally.
