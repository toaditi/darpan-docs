title: "Connections"
description: "Troubleshoot external connection records."
---

## SFTP

Check host, port, username, authentication method, directory path, and network access. Confirm whether the server expects a private key, password, or another authentication method.

## NetSuite

Check account target, role permissions, token validity, integration status, script deployment, and SuiteQL field names.

## AI providers

Check the provider record, credential reference, model availability, and feature-specific limits.

## Safe debugging

Capture error codes and messages, but redact secrets and customer-specific identifiers before sharing externally.
