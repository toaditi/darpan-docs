## SFTP

Search **Ask Darpan** for **Open SFTP Servers**. For an existing record, search by SFTP description, host, or username. Check host, port, username, authentication method, directory path, and network access. Confirm whether the server expects a private key, password, or another authentication method.

## NetSuite

Search **Ask Darpan** for **Open NetSuite Settings**. Check account target, role permissions, token validity, integration status, script deployment, and SuiteQL field names.

## AI providers

Search **Ask Darpan** for **Open AI Settings**. Check the provider record, credential reference, model availability, and feature-specific limits.

## Safe diagnostics

Capture error codes and messages with secrets and organization-specific identifiers redacted before sharing externally.
