---
title: "Configure AI providers"
description: "Set up AI provider records used by assisted Darpan workflows."
---

Darpan can use saved AI provider settings for assisted workflows such as schema support or analysis features.

## Setup checklist

1. Open **Ask Darpan** with `Cmd/Ctrl+K` or the floating **Ask Darpan** button.
2. Search for **Open AI Settings**.
3. Select **Open AI Settings**.
4. Create or edit the provider record.
5. Enter the provider identifier and required credential reference.
6. Save the record.
7. Run the smallest workflow that uses the provider.

## Good records

Use clear names and descriptions. A future user should be able to tell whether a provider is for local testing, pilot use, or a production-like target.

## Safety

Do not document API keys, organization identifiers, or private model routing details in this public docs repo.
