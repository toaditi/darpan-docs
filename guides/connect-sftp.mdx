---
title: "Connect SFTP"
description: "Create or maintain an SFTP connection record."
---

Use an SFTP connection when a run reads input files from, or writes output files to, an external server.

## Before you start

Confirm these values outside the docs site:

- Host and port
- Username
- Authentication method
- Root or working directory
- Expected input and output folders

## Create the connection

1. Open **Ask Darpan** with `Cmd/Ctrl+K` or the floating **Ask Darpan** button.
2. Search for **Open SFTP Servers**.
3. Start the create workflow from the SFTP surface.
4. Enter the connection name and server details.
5. Save the record.

## Edit an existing connection

1. Open **Ask Darpan**.
2. Search for the SFTP description, host, or username.
3. Select the **Edit SFTP: ...** result.
4. Update the record and save it.

## Validation

After saving, run the smallest workflow that proves the server path and credentials work. If validation fails, check path, authentication method, and network access first.

<Warning>
Never paste private keys, passwords, or customer hostnames into public docs.
</Warning>
