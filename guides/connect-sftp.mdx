---
title: "Connect SFTP"
description: "Create or maintain an SFTP connection record."
---

Use SFTP connections when Darpan needs to exchange files with an external server.

## Before you start

Confirm these values outside the docs site:

- Host and port
- Username
- Authentication method
- Root or working directory
- Expected input and output folders

## Create the connection

1. Open Darpan.
2. Go to **Settings**.
3. Open **SFTP**.
4. Start the create workflow.
5. Enter the connection name and server details.
6. Save the record.

## Validation

After saving, run the smallest workflow that proves the server path and credentials work. If validation fails, confirm the path, authentication method, and network access first.

<Warning>
Never paste private keys, passwords, or customer hostnames into public docs.
</Warning>
